# PROJET MOBILE LUCAS AMBIEHL 4A

Création d'une application mobile affichant une liste d'éléments ainsi que leur détails

## Notation

### Notions abordées 

* Architecture *MVVM*
* Clean Architecture
* Liste affichée à l'aide d'une *RecyclerView*
* Design fonctionnel
* Travail sur Koin et Room
* Features
* Tests unitaires
